
第一次诺亚泰拉大迁徙浪潮（Noah Terra Migration Wave I）

2112年，第一次诺亚泰拉大迁徙浪潮结束，火星常驻人口总数达到5亿。
当然，5亿对于火星的城市化规模来说并不拥挤，仍颇为地广人稀。

在第一次诺亚泰拉大迁徙浪潮中，火星人口的管理最初仍是以各国分治的形式进行，即各国独立自己的办事处，并尽可能地将本国人口聚集。

但由于诺亚泰拉栖居倡议本身出自于HCA，这导致移民手续及后续的管理如果以严格的国际间合作方式进行会造成巨大的繁文缛节，加之火星的管辖本身已经跨越地界，来到统一的地外空间，这使得很多人认为这不应该沿用地球的管理与治理方式。
随着火星常驻人口增多且民众呼声越来越高，而火星精英化的社群又通常在背后拥有一定的话语权和权力，这使得火星的独立管制势在必行。

2112年，地球方面在HCA的协调下，同意在火星建立统一的政治实体，而其性质是一种国际间组织，即不具有国家主权，但拥有行使等效主权的权利；而该组织直接隶属于HCA，由此也象征着HCA打破了部分开山立下的戒律——开始染指政治，并进入一个崭新的“文明统治”时代。

第一次诺亚泰拉大迁徙浪潮结束后，地球方面普遍认为火星的精英化长此以往会导致严重的社会问题，加剧阶级分化、贫富差距并由于地火难以具有地理上的联系，这种问题会愈发难以调和，另一方面，精英化容易造成独裁统治局面，尽管目前HCA的原则仍有其效力和影响力，但未来的风险不可预估。
地球方面开始筹划第二波人类迁徙计划，但这次计划的实施难度远比想象中复杂。
